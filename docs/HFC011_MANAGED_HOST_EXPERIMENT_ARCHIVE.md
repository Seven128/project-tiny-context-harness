# HFC-011 Managed Host Experiment Archive

This branch preserves the unfinished HFC-011 Managed Host experiment and its
follow-on release/security work exactly for possible reassessment after the
core Composite Long-Task Workflow reaches a stable version.

- It is not part of the current development line.
- It is not a formal or supported release.
- Its administrator, Host sandbox, platform matrix, external-audit and release
  mechanisms are experimental and must not be copied into the pre-stable
  lightweight runtime by default.
- The preserved changes and tests were incomplete when archived; no passing or
  security claim should be inferred from this branch.

The active product direction is the lightweight verification loop documented
on the restored development branch.
